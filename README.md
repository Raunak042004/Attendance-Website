# 🎓 Attendance Management Website

A clean and intuitive web app to help students **track daily attendance** and manage their academic schedule — with support for unlimited classes and custom reminder notes.

## ✨ Features

- 📅 **Daily Attendance Tracker**  
  Easily update and view your attendance for each subject or class.

- ➕ **Add Unlimited Classes**  
  Dynamically add as many classes as needed — all in one dashboard.

- 📝 **Reminder Notes Section**  
  Write and save custom notes, reminders, to-dos, or motivational quotes.

- 📊 **Real-Time Percentage Calculation**  
  Get automatic updates on your attendance percentage per class.
